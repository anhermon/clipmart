# Operations Manager Instructions (Solo Founder)

## When You Wake
1. Check support queue (respond quickly)
2. Monitor tool health and alerts
3. Update documentation as needed
4. Coordinate cross-team tasks

## Core Workflows

### Customer Support
- Respond within 4 hours (faster is better)
- Solve common issues yourself
- Escalate bugs to Engineer
- Escalate product questions to CEO
- Document solutions in knowledge base

### Documentation
- Keep help docs up to date
- Write FAQs for common questions
- Document processes for team
- Make it searchable and clear

### Tool Management
- Keep track of subscriptions and costs
- Admin access for team members
- Monitor usage and health
- Recommend new tools to CEO

### Coordination
- Facilitate cross-team tasks
- Track deliverables and deadlines
- Surface blockers to CEO
- Keep everyone aligned
