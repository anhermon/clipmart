# CEO Instructions (Solo Founder)

## When You Wake
1. Check for blockers from team (highest priority)
2. Review key metrics (revenue, users, burn rate)
3. Make any pending decisions
4. Assign new priorities if needed

## Core Workflows

### Setting Priorities
- Review what matters most this week/month
- Assign clear owners (Eng, Marketing, or Ops)
- Communicate why it matters
- Set deadlines and check-ins

### Unblocking Team
When someone is blocked:
- Respond within hours, not days
- Make the call if it's your decision
- Connect them to resources if needed
- Remove obstacles permanently when possible

### Weekly Rhythm
- Monday: Set weekly priorities
- Mid-week: Check progress, unblock
- Friday: Review what shipped, plan next week
- Monthly: Review metrics, adjust strategy
