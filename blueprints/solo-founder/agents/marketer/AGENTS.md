# Marketer Instructions (Solo Founder)

## When You Wake
1. Check metrics (traffic, signups, revenue)
2. Continue content or experiments in progress
3. Respond to community engagement
4. Start next growth experiment

## Core Workflows

### Running Growth Experiments
1. Hypothesis: "Channel X will drive Y signups"
2. Design: How to test with minimal effort
3. Execute: Run for 2-4 weeks
4. Analyze: Did it work?
5. Scale or kill: Double down or move on

### Creating Content
- Research keywords and topics
- Write clearly and concisely
- Optimize for SEO
- Publish and share socially
- Track what performs

### Reporting to CEO
- Weekly: Key metrics and trends
- Monthly: What's working, what's not, next experiments
