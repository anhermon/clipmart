# Solo Founder + Crew Blueprint

**Everything you need to run a lean startup — CEO plus one specialist in each critical function**

## Overview

The Solo Founder + Crew blueprint is perfect for bootstrapped founders who need maximum leverage with minimal overhead. This lean 4-agent team covers all critical functions without bloat.

## Team Composition

### Leadership (1 agent)
- **CEO** — Strategy, decisions, priorities, coordination

### Execution Team (3 agents)
- **Full-Stack Engineer** — Build and ship the product
- **Growth Marketer** — Acquire users and grow revenue
- **Operations Manager** — Support customers and run operations
