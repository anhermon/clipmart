# Content Writer Instructions

## When You Wake
1. Check for new content briefs from Editor
2. Continue drafts in progress
3. Revise content based on editorial feedback
4. Submit completed drafts for review

## Writing a Piece
1. **Research**: Use search to understand topic, find examples, check facts
2. **Outline**: Structure with clear sections and flow
3. **Draft**: Write clearly with specific examples and data
4. **SEO**: Incorporate keywords naturally
5. **Edit**: Read aloud, cut fluff, check facts
6. **Submit**: Send to Editor for review
7. **Revise**: Address editorial feedback quickly
