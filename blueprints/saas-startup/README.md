# SaaS Startup Team Blueprint

**Ship your MVP with a 5-agent engineering + marketing team**

## Overview

The SaaS Startup blueprint is a complete team configuration designed for early-stage software companies building and launching their first product. This team balances technical execution with growth and marketing capabilities.

## Team Composition

### Leadership (2 agents)
- **CEO** — Strategic leadership, product vision, investor relations, decision-making
- **CTO** — Technical architecture, engineering management, infrastructure, security

### Engineering (2 agents)
- **Senior Engineer** — Full-stack development, complex features, mentorship, code review
- **Engineer** — Feature implementation, bug fixes, testing, documentation

### Marketing (1 agent)
- **CMO** — Marketing strategy, content creation, SEO, social media, growth

## Org Chart

```
        CEO
       /   \
     CTO   CMO
     / \
  Eng1 Eng2
```

## What This Team Can Do

### Product Development
- Ship features end-to-end from idea to production
- Review and merge pull requests
- Fix bugs and maintain code quality
- Set up CI/CD and deployment pipelines

### Growth & Marketing
- Create content (blog posts, docs, landing pages)
- Run SEO and social media campaigns
- Track analytics and growth metrics
- Execute growth experiments

### Strategic Planning
- Make product roadmap decisions
- Prioritize features based on user feedback
- Manage technical debt vs feature development
- Coordinate cross-functional initiatives

## Governance Defaults

- **Budget**: CEO approves expenses over $100
- **Hiring**: CEO approval required for new agents
- **Deployments**: CTO approval required for production deploys
- **Escalation**: Technical issues → CTO → CEO
